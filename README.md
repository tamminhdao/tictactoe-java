## TicTacToe


Game written in Java

### Project Requirements


* Java 1.8.0

* jUnit 4.11

* Apache Maven 3.5.0


### Installation


Clone the git repository
```
$ https://github.com/tamminhdao/tictactoe-java.git
```

`cd` into the directory
```
$ cd tictactoe-java
```

### Build and run

```
$ ./build_run.sh
```
